선착순 1,000명 데이터 더블업(5GB * 2) 이벤트 유의사항

  1. 1. 해당 이벤트는 선착순 1,000명에 한 해 적용됩니다.
  2. 2. 선착순 1,000명은 개통완료 기준입니다. *신청 완료 기준이 아닙니다.
  3. 3. 데이터 제공에 대한 유의사항은 [데이터 추가제공 (5GB) 공통 유의 사항]을 참고하시기 바랍니다.
  4. 4. 선착순 ,000명 마감시 [데이터 더블업(5GB * 2)] 이벤트는 자동 종료 되며, [데이터 추가제공 (5GB)] 이벤트가 적용됩니다.
  5. 5. 슈가칩 11GB+매일2GB, Sugar 11GB + 매일 GB(12개월+) 상품은 2024년 09월 06일 가입자부터 선착순에 포함됩니다.

풍성한 9월 데이터 추가제공 이벤트 유의 사항

  1. 1. 본 요금제는 2024년 09월 01일 ~ 09월 30일까지 개통 완료한 가입자에 한해 혜택이 적용됩니다.
  2. 2. 본 요금제는 9일 의무 사용 기간이 있으며, 의무 사용기간 내 해지 시 통신사 정책에 따라 위약금이 발생합니다. * 위약금 산정 방식 : 사용기간 동의 합입한 금액
  3. 3. 가입 월의 기본료 및 기본 제공량은 개통일 기준 일할 산정 됩니다.
  4. 4. 본 요금제의 할인 혜택은 개통 시에만 적용되며, 기존 가입자의 요금제 변경 시에는 적용되지 않습니다.
  5. 5. 이름 중 요금제 변경 및 명의 변경할 경우 할인 혜택이 종료됩니다.
  6. 6. 일시 정지 시에는 할인 혜택도 일시 중단되며, 할인 기간에 정지 기간이 포함됩니다. * 일시 정지 기본료 4,400원이 청구됩니다.
  7. 7. 타 이벤트에 중복으로 참여할 수 있습니다.
  8. 8. 해당 이벤트는 조기 마감 및 통신사 사정에 의해 예고 없이 종료될 수 있습니다.
  9. 9. 슈가헬 11GB+매월2GB, Sugar 11GB+매월2GB(12개월+) 상품은 2024년 09월 06일 가입자부터 데이터 추가제공 이벤트에 포함됩니다.

데이터 추가제공(5GB) 공통 유의 사항

  1. 1. 데이터 소진 순서 요금제 기본 제공 데이터 > 추가 제공 데이터 > QoS 데이터
  2. 2. 추가 제공 데이터는 가입 당월의 경우 매주 금요일 제공됩니다. (지난주 금요일 ~ 이번 주 목요일 가입 대상) * 가입 익월부터는 매월 1일 제공
  3. 3. 추가 제공 데이터는 일시 정지, 요금제 변경, 명의 변경 시 자동 제공 종료되며, 원 상태로 복원돼도 다시 제공이 불가능합니다.
  4. 4. 추가 제공 데이터는 전량 테더링으로 사용할 수 있습니다.
  5. 5. 추가 제공 데이터는 이월되지 않습니다.
  6. 6. 추가 제공 데이터는 로밍용으로 사용할 수 없습니다.
